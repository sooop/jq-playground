

## 개선사항

### 자동완성 기능 개선

- Input Data의 구조와 문맥을 감지하는 지능형 자동완성
- 특히 jq의 파이프 문법으로 키 레벨을 더 깊이 내려갔을 때에도 문맥 유지 필요
- tab 키를 사용하여 자동완성 항목 순환


### 쿼리 자동실행 개선

- 일반적으로 '재생중'일 때 UI는 일시정지 버튼으로 표현함. 따라서 자동실행 on/off 일 때 표시되는 아이콘 형태 교체
- 자동실행 중일 때에는 Input 패널 헤더에 Chip UI로 알림 표현
- 데이터의 크기가 매우 큰 경우에는 쿼리 편집으로 반복 실행하면서 매우 느려질 수 있음. 특정한 크기 이상의 데이터가 입력되면 자동 실행 기능 중지
- 쿼리 변경 시 자동 실행이 트리거되는 디바운스 시간 체크하고 보다 현실화하기

### 기타 UI 관련 수정 사항

- 창크기 조절한 내역을 local storage에 저장하여 기억하기
- 그 외에 단일 스레드로 인한 병목을 개선할 수 있는 방법이 있는지 모색하기
